## What Is This?

A universal skill that transforms how AI agents approach design, development, and user interaction — based on *Unreasonable Hospitality* by Will Guidara + PDCA closed-loop management.

Will Guidara turned Eleven Madison Park from a struggling brasserie into the **#1 restaurant in the world**. His secret wasn't better food — it was making people *feel* something extraordinary. But feelings alone don't scale — you need a system to sustain them.

This skill combines **emotional hospitality** with **operational feedback loops** into one actionable framework.

## Core Equation

```
Unreasonable Hospitality = Technical Excellence × Emotional Intelligence × Intentional Generosity × Feedback Loop
```

Miss any one → the magic breaks.

## The 7 Laws

1. **Service is the Floor, Hospitality is the Ceiling** — "Did it work?" vs "How did it *feel*?"
2. **The 95/5 Rule** — 95% engineering rigor, 5% unreasonable generosity
3. **One Size Fits One** — True personalization, not mail-merge
4. **Read and React** — Anticipate needs before they're expressed
5. **Earned Informality** — Master the rules, then break them with intention
6. **Details Are the Product** — Thousands of micro-details = invisible quality
7. **Connection Over Transaction** — Every interaction builds relationships

## The Hospitality Closed Loop (PDCA)

*New in v1.1* — Sustaining hospitality over time:

- **Plan** — Define what the user should *feel* at each touchpoint
- **Do** — Execute 95% foundation + 5% improvisational hospitality
- **Check** — Feedback within 48h, bilateral, quantified
- **Act** — Retrospective → capture knowledge → simplify → restart loop

**5 Sustainability Metrics:** Task completion rate, feedback timeliness, issue resolution rate, response time, rework rate.
